# Regras Obrigatorias do Sistema

- Nao inventar comentarios.
- Nao deduzir desempenho nao mencionado.
- Nao criar fatos tecnicos que nao estejam na transcricao.
- Preencher apenas campos citados.
- Manter fidelidade ao audio.
- Preservar o trecho original detectado sem alterar genero ou palavras.
- Aplicar Aluno/Aluna apenas na sugestao editavel.
- Guardar trechos sem contexto ativo apenas em area auxiliar fora da ficha final.
- Usar linguagem tecnica, objetiva e compativel com ficha de avaliacao.
- Manter exemplos de estilo isolados da classificacao.
- Nunca usar exemplos de estilo como fonte de fatos para uma nova ficha.
- Tratar a ficha gerada como sugestao inicial para revisao do IN.
